# Bibliotheksverwaltung

Einfache Spring Boot Anwendung zur Verwaltung einer Bibliothek. Dient als Beispielprojekt zum Testen von Code-Reviews.

## Technologie

- Java / Spring Boot
- Spring Data JPA
- H2 In-Memory Datenbank
- REST-API (kein GUI)

## Starten

```bash
mvn spring-boot:run
```
